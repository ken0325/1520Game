# 猜拳1520遊戲

## Overview

這是一款用Java開發的安卓遊戲，名為猜拳1520。玩家將與電腦進行對戰，並透過個人技巧和運氣進行猜拳遊戲。遊戲以"1520"（十五二十）的規則進行，玩家同時出拳並猜測所有玩家的出拳總和，目標是避免猜錯或成為最後一個猜的人，這通常會導致懲罰。

## Features

- **與電腦對戰**: 玩家可以選擇出拳(分別左右手)（0到5根手指），與電腦進行比賽。
- **API集成**: 使用API獲取電腦的隨機出拳，以增強遊戲的隨機性和挑戰性。
- **結果記錄**: 利用SQLite數據庫存儲每次對戰的結果，記錄玩家與電腦的出拳及其結果。

## Technologies Used

- **Programming Language**: Java
- **Platform**: Android
- **APIs**:
  - 隨機出拳API，提供電腦出拳數據。
- **Data Format**: JSON
- **Database**: SQLite
